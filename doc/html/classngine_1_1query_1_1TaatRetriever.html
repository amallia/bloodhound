Detailed Description

template<typename PostingList, bool prefetch = false, unsigned short init_gap = 0, unsigned int acc_block_size = 0>
class ngine::query::TaatRetriever< PostingList, prefetch, init_gap, acc_block_size >

Term-at-a-time document retriever.

PostingList requires:

  • doc_begin() and doc_end(): document start and end iterator, respectively;
  • score_begin() and score_end(): score start and end iterator, respectively;
  • length(): posting list length;
